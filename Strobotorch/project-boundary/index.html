Project boundary

Strobotorch Is Independent Android Tooling

Strobotorch is about torch and camera APIs, visual signal encoding, visual signal decoding, diagnostic capture, and safety documentation around bright flashing light.

Brain Candy boundary

Strobotorch is separate from Till Holzapfel's scientific-advisor work for Brain Candy. The app covers Android torch/camera tooling, visual signal tests, and safety documentation.

Tooling scope

Strobotorch is an open-source Android tooling project for public torch APIs, camera/luma diagnostics, torch event recording, visual signal encoding, visual signal decoding, and device-limit documentation.

Use exclusions

Strobotorch excludes state-shift induction, hallucination induction, wellness, medical, meditation, brainwave-entrainment, and clinical use.
