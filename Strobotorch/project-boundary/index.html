Project scope

Strobotorch Is Android Signal Tooling

Strobotorch is an Android flashlight signaling and diagnostics tool for torch and camera APIs, visual signal encoding, visual signal decoding, diagnostic capture, and safety documentation around bright flashing light.

Scope

Strobotorch is independent from Brain Candy. It is not a hallucination tool, wellness device, medical device, meditation product, brainwave-entrainment system, or state-shift platform.

Tooling scope

Strobotorch is an open-source Android tooling project for public torch APIs, camera/luma diagnostics, torch event recording, visual signal encoding, visual signal decoding, and device-limit documentation.

Use exclusions

Strobotorch excludes state-shift induction, hallucination induction, wellness, medical, meditation, brainwave-entrainment, and clinical use.
